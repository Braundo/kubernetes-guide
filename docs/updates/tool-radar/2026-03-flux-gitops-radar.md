# Flux Tool Radar (GitOps for Kubernetes)

Flux is a mature GitOps framework focused on declarative delivery and reconciliation for Kubernetes clusters.

## What the Tool Does

Flux continuously reconciles cluster state from Git, with modular controllers for source management, kustomization, image automation, and Helm delivery.

## Why It Is Worth Watching

Flux remains a strong choice for teams that want a composable GitOps stack with deep Kubernetes-native integration and CNCF ecosystem alignment.

## Maturity and Adoption Notes

Flux is stable and widely used. Teams should define clear multi-environment repository strategy and controller ownership from the beginning.
