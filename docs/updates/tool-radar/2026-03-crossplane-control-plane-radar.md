# Crossplane Tool Radar (Cloud Control Plane APIs)

Crossplane allows platform teams to expose infrastructure capabilities as Kubernetes APIs, enabling internal platform products with policy and composability.

## What the Tool Does

Crossplane manages cloud resources through CRDs and compositions. Teams can define higher-level abstractions so application teams consume standardized APIs instead of raw cloud accounts and scripts.

## Why It Is Worth Watching

For internal platform engineering programs, Crossplane can become the contract layer between application teams and infrastructure teams. It is especially useful when standardizing multi-cloud provisioning patterns.

## Maturity and Adoption Notes

Crossplane is established and growing, with strong ecosystem activity. Adoption requires solid platform governance and API lifecycle ownership to avoid CRD sprawl.
