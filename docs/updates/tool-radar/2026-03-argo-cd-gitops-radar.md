# Argo CD Tool Radar (GitOps Delivery)

Argo CD remains one of the dominant GitOps delivery tools for Kubernetes platform teams.

## What the Tool Does

Argo CD continuously reconciles Kubernetes resources from Git and provides deployment state visibility, rollback workflows, and multi-cluster delivery controls.

## Why It Is Worth Watching

GitOps continues to be the default operating model for many Kubernetes platforms. Argo CD offers high flexibility for multi-team delivery and policy-driven environment promotion.

## Maturity and Adoption Notes

Argo CD is mature and broadly adopted. Operational complexity can increase at scale, especially around application topology and controller sizing, so platform standardization is important.
