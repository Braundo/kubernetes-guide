# Recommended Kubernetes Resources

This page is a curated resource shelf for Kubernetes operators, platform engineers, and SRE teams.

## Kubernetes and Cloud-Native Books

| Resource | Why it is useful |
| --- | --- |
| [Kubernetes: Up and Running (3rd ed.)](https://www.amazon.com/s?k=Kubernetes+Up+and+Running+3rd+Edition&tag=k8sguide01-20) | Practical foundation for architecture and operations |
| [Kubernetes in Action (2nd ed.)](https://www.amazon.com/s?k=Kubernetes+in+Action+Second+Edition&tag=k8sguide01-20) | Deep walkthrough of real-world platform usage |
| [Production Kubernetes](https://www.amazon.com/s?k=Production+Kubernetes+book&tag=k8sguide01-20) | Reliability and production operations patterns |
| [Cloud Native DevOps with Kubernetes](https://www.amazon.com/s?k=Cloud+Native+DevOps+with+Kubernetes&tag=k8sguide01-20) | CI/CD and platform workflows on Kubernetes |
